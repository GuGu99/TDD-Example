# TDD-Example
Node.js API Server Testing with Mocha + Should + SuperTest

## Run
Start This Server

```bash
$ npm start
```

Monitoring Server with [Nodemon](https://www.npmjs.com/package/nodemon)

```bash
$ npm restart
```

## Test
API Test [(src/api/users/users.spec.js)](https://github.com/GuGu99/TDD-Example/blob/master/src/api/users/users.spec.js)

```bash
$ npm test
```
